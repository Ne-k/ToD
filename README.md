# ToD

To run the bot just make a `.env` file and fill out the following.

```
Token="Bot token"


Fluxpoint_API_Key="fluxpoint.dev API key"
----------------------------------------------
GUILD_LOGGING_ID="Webhook ID"
GUILD_LOGGING_WEBHOOK="webhook token"
----------------------------------------------
CLIENT_LOGGING_ID="webhook id"
CLIENT_LOGGING_WEBHOOK="webhook token"
----------------------------------------------
SHARD_LOGGING_ID="webhook id"
SHARD_LOGGING_WEBHOOK="webhook token"
----------------------------------------------
SUGGEST_LOGGING_ID="webhook id"
SUGGEST_LOGGING_WEBHOOK="webhook token"
----------------------------------------------
OTHER_LOGGING_ID="webhook id"
OTHER_LOGGING_WEBHOOK="webhook token"
===================================================
KAEDE_CDN_KEY=" kaedee.xyz API key"
MONGOSTRING="Mongo DB string"

```
